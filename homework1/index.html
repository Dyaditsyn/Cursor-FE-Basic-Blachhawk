Base homework 1

  • git config --global user.name blabla - To set your user name in the main configuration file.
  • git config --global user.email blabal@bla - To set your email in the main configuration file.
  • git init - To initialise a git repository for a new or existing project.
  • git status - To check the status of files since your last commit
  • git add - adds changes in your working directory.
  • git commit - commits your changes and sets it to new commit object for your remote.
  • git log - Show commit logs
  • git checkout - Switch to different branches
  • git branch - Lists out all the branches.
  • git push - Push your changes to remote.
  • git pull - Pull your changes to remote.
  • git merge - Merge two branches you were working on.
  • git clone - To copy a git repository from remote source.

.gitignore file - a text file that tells Git which files or folders to ignore in a project.
